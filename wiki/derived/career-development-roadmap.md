# Career Development Roadmap: 200K to 300–400K

> **TL;DR**: A concrete 8-month plan for a senior ML engineer to reach Staff-level TC (300–400K SGD) by combining GenAI technical depth, public portfolio building, and targeted networking — the gap is not technical capability but visibility and packaging.

## Key Points

- **Core insight**: the 200K→400K gap is not technical skill but impact visibility — same work done privately vs. publicly documented with business metrics is valued 2× differently
- **Phase 1 (Month 1–2)**: consolidate GitHub into 3 public flagship repos — `scientific-paper-agent` (multi-agent + RAG + multi-modal), `ml-system-design` (personal rewrites of 21 case studies with LLM integration angle), `rag-experiments` (naive vs. advanced RAG with A/B eval data)
- **Phase 2 (Month 2–5)**: 5-article writing plan targeting hiring managers — (1) Recommendation system × LLM reranking, (2) Claude Code leak → 7 production agent patterns, (3) RAG A/B test data from scientific extraction, (4) Multi-agent system design with memory and compression, (5) ML-to-GenAI transition retrospective
- **Phase 3 (Month 5–8)**: simultaneous interviews at 3–5 companies with competing offers; negotiate at Staff level (not just Senior TC) — refresher stock in years 2–3 is worth more than sign-on
- **Differentiator angle**: traditional ML (recommendation, search, ads ranking) + GenAI (Agent, RAG, fine-tuning) cross-domain knowledge — interviewers rarely meet candidates who understand both infra and LLM reasoning
- **Scientific Paper Agent as centerpiece**: the flagship project links to ACE architecture, multi-agent patterns, tiered memory, multi-modal extraction, and domain-specific fine-tuning — covers every major Staff-level AI interview topic in one coherent system

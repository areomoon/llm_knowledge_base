# OpenAI Plans Desktop Superapp Merging ChatGPT, Codex, and Atlas

> **TL;DR**: OpenAI is consolidating ChatGPT, Codex, and its Atlas browser into a single desktop superapp — mirroring Anthropic's bundled Claude strategy — signaling a broader industry shift from API-driven development to agent-orchestrated superapp interfaces.

## Key Points

- OpenAI plans to merge ChatGPT, Codex, and Atlas Browser into one desktop superapp; Fidji Simo (CEO of Applications) is leading the effort
- In March 2026, Claude overtook ChatGPT as the top US app-store download after Anthropic bundled Claude chatbot + Code + Cowork into a unified app
- The superapp strategy centers on agentic AI: autonomous multi-step task execution across tools within a single application surface
- This signals a paradigm shift — future software should be designed as "agent-friendly tools" rather than traditional user-facing UIs
- Anthropic's MCP (Model Context Protocol) is the emerging standard enabling this pattern: apps expose tool interfaces consumed by agents, not humans directly

# OpenAI Plans Desktop Superapp Merging ChatGPT, Codex, and Atlas

OpenAI plans to merge ChatGPT, Codex, and Atlas Browser into a desktop superapp. Fidji Simo (CEO of Applications) is leading the change. The superapp emphasizes agentic AI capabilities - autonomous task execution across tools. In March 2026, Claude overtook ChatGPT as the top US download; Anthropic already bundled Claude chatbot + Code + Cowork into one app, which OpenAI is now mirroring.

Key user insight: This signals a paradigm shift from traditional API-based development to SuperAgent interfaces. Future apps should be designed as "agent-friendly tools" rather than traditional user-facing UIs. MCP (Model Context Protocol) by Anthropic is already enabling this pattern. Apps become tool interfaces for agents, not direct user interfaces.

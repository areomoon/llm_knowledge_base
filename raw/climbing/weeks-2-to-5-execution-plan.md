## 🎯 Breadth-first framework（2026-04-22 新增）

**「V6-V7 level」的 operational definition**：
- **V6 level** = 一天內完攀 **3 條不同風格** V6（fingery 垂直 / 技術板 / 輕仰
  角 / 強仰角 / compression 任選 3）
- **V7 attempt-ready** = 具備對符合強項的 V7 做 3-4 次 quality attempt 的能力
  （不鎖定特定路線 send）

**為何捨棄「Laputa send」單點目標**：
1. 原計劃 Sat 全押同一條 project = project mode，非 level-up mode；對整體水
   平提升 ROI 低
2. Laputa / LAST CALL 都是 compression / deep-lock → 正好壓在
   bicep/brachialis chain 上（目前最受限資源）
3. Level = breadth（風格橫跨）+ base（指力 + PE + 拮抗）；Flash bucket 權
   重拉高，Sat 從 "project day" 改為 "breadth verification day"

## 核心原則（4 週貫穿）

1. **指力優先**（Lattice 共識，V5-V10 #1 limiter）
2. **每減 1kg ≈ +1.5% 有效指力**（Lattice *Stay Light or Get Strong?*）

4. **Rest more than you think**（Adam Ondra *The Nugget* Playlist #38）
5. **Progressive overload**：每週 +5% 負重或 +1 組，不 linear 就 plateau

## Week 2 — 建立 Baseline + PE 啟動 (2026-04-20 ~ 04-26)

### 新增 vs Week 1
- Monday 開始每週一次 **4×4 PE session**
- Thursday 補做 **hangboard baseline 測試**
- 每日加 **creatine 5g + Vitamin D3 2000 IU + Omega-3 2g**
- 每週 2 次（Thu + Sat 前）**collagen 15g + VitC 50mg** 於訓練前 30-45 min

### Monday 4/20 — Volume + 4×4 PE

**Volume 部分（60 min）**
- 熱身 + 暖身路線 15 min
- 攀 V3-V4 共 10-12 條（不需完攀每條，累計量）
- 休息 15-20 min

**4×4 PE 部分（25 min）**
- 選 4 條 V3-V4 難度（你的 flash-2 grade）
- 4 條連做 4 輪，**每輪之間只休攀爬時間 1:1**
- 輪與輪之間休 2-3 min
- 目標：**完成 16 次攀爬不掉落超過 2 次**

> 依據：Lattice *Training Climbing Power Endurance - 3 Mistakes* (Playlist #52)
> — 強調 PE 訓練必須 incomplete rest，不是 complete rest

### Thursday 4/23 — Hangboard Baseline + 輕量攀岩

**Hangboard max hang 測試（30 min）**
- 熱身：反覆在 20mm edge 上 bodyweight hang，6-10s × 3 組，逐漸適應
- Max hang protocol（Emil Abrahamsson）：
  - 20mm edge, half-crimp
  - 10 秒單次 max hang，休 **5 分鐘**
  - 逐漸 +2.5kg 至「能撐 8 秒但 10 秒掉」 → 該重量即為 baseline max
- 記錄最大負重（例：體重 + 10 kg = 82 kg total）

**之後輕量攀岩（30-45 min）**
- V3-V4 technique drill（silent feet、flag、drop knee 專注）
- 不攀高難度，手指已用

### Saturday 4/25 — Breadth Day 1: V3-V5 多風格 Volume

**2026-04-22 改寫**：原 "V5 flash → V6 project → V7 嘗試" 的 peak 模式取消 ——
本週 Thu 04-23 hangboard 為條件式 + bicep chain 急性過載修復窗，Sat 不適合
peak intensity。改為 **breadth volume day**，目標是在多風格橫掃 V3-V5，
為 Week 3 flash bucket 重啟鋪路。

**流程**：
1. 起床：水 500ml（**不吃 collagen**，今日無 peak loading）
2. 早餐：碳水 60-80g + 蛋白 20-30g，訓練前 **2 hr 停食**
3. 攀岩前 30-45 min：咖啡 **2 mg/kg = 140-150mg**（比 peak day 半量，維持 alert
   不推 CNS 到底）
4. 熱身 20-30 min（含 bodyweight dead hang recirculation）
5. 主體：
   - **V3-V4 flash 5-7 條**（多風格，每風格 1-2 條）
   - **V5 flash 3-5 條**（max 2 go per 條，不堆 attempts）
   - **禁止 V5 project（≥ 3 go 即放棄）** —— 本日核心紀律
   - **禁止任何 compression / deep-lock / 深仰角**（bicep chain 保護期）
6. **不攻 V6+，不摸 Laputa / LAST CALL**
7. 攀岩後 30 min 內：30g protein + 60-80g carbs

**KPI（本日記錄）**：
- [ ] 完攀 V3-V5 總條數：______ / 10-15
- [ ] 涵蓋風格數：______ / 4+（fingery 垂直 / 技術板 / 輕仰角 / 強仰角 / 平衡 / slab 任選）
- [ ] 攀岩中 bicep/brachialis 牽扯感：無 / 輕微 / 明顯
- [ ] 前臂 pump 最高點：______ / 10
- [ ] 體力剩餘感（攀完後）：______ / 10

**判讀規則**：
- 完攀 10-15 條 + 4 風格 + 無 bicep 訊號 → **bicep chain 清零窗 day 1 達成**，
  Week 3 Mon 可回 Volume + 4×4 PE
- 完攀 ≥ 8 條 + 3 風格 + 無 bicep 訊號 → 同上（最低達標）
- 任一 bicep 訊號出現 → **chain 清零窗 reset**，Week 3 Mon 再降為 volume only
  （無 4×4）；本週期 V7 attempt-ready 目標時程需重議

### 其他日
- Tue 4/21: 下肢（pistol squat, goblet squat, lunge）+ core + 拮抗 30-40 min
- Fri 4/24: Lock-off + compression（本週五 template）
- Wed 4/22 + Sun 4/26: 視 Mon/Thu 恢復狀況；**Wed 可以再攀**若感覺好，**Sun 必須休**

## 飲食執行表（每日）

### 攀岩日 / 訓練日

| 時段 | 內容 | 目的 |
|-----|------|-----|
| 起床 | 水 500ml + 若訓練在 early：collagen 15g + VitC 50mg | 肌腱、hydration |

| 訓練前 2hr | **停食**（避免攀岩時胃不適） | — |
| 訓練前 30-45min | 咖啡 3-5mg/kg（僅 Sat peak + 重要 hangboard 日） | 爆發力 + 疼痛閾值 |
| 訓練前 30-45min | collagen 15g + VitC 50mg（Thu + Sat） | 肌腱 collagen synthesis |
| 訓練中 | 水 + 少量 electrolyte（流汗多時） | 維持 performance |
| 訓練後 30min 內 | protein 30g + carbs 60g（奶昔 + 香蕉、或正餐） | 肌肉恢復窗 |
| 晚餐 | 蛋白 30-40g + 蔬菜 + 碳水 40-60g | 每日 125-145g 蛋白目標 |
| 睡前 1hr | casein / cottage cheese 20g（可選） | 夜間肌肉合成 |

### 休息日

- 碳水降到 2-3g/kg（= 150-220g），蛋白維持 125g+
- 總熱量 1700-1900 kcal（比訓練日 -200 kcal，創造 deficit）
- 仍吃 creatine + D3 + Omega-3
- 不吃 collagen + VitC（僅訓練前有意義）

### 每日 supplement

| 補充品 | 劑量 | 時機 | 為何 |
|-------|-----|-----|------|

| Vitamin D3 | 2000-4000 IU | 早餐隨脂肪 | SG 室內工作者幾乎缺 |

## Deload 訊號（出現任一就減量）

- 指關節腫脹 / 僵硬 >24 hr
- 前臂酸痛 >48 hr 未退
- 睡眠品質突降 (<6 hr 連續 2 晚)
- 心跳晨起 >+10% baseline
- 任一手指刺痛

**處理**：立即休 2-3 天完全不碰手指；補眠 + 補水 + 蛋白質。

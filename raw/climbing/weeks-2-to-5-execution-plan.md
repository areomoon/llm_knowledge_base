# Weeks 2-5 Execution Plan

> 基於 Week 1 實際執行（Mon-Wed-Sat 攀岩節奏 + 04-15 session 暴露的四大 limiter）+
> authority-cited 訓練科學（Lattice、Hooper's Beta、Emil Abrahamsson、Keith Baar、
> Adam Ondra）改寫的 4 週 peak 向計劃。

---

## 本計劃假設前提

- 目標：**5/17 前穩定 V6 + send 一條 V7**（如 `LAST CALL` 或 `LAPUTA`）
- Profile：32M / 173cm / 72kg / 引體 20 下 / 4 年經驗
- 已知 limiters（04-15 session 驗證）：指力專項（非 axial + pinch）、左側 lock-off
  asymmetry、compression pull、power endurance
- 實際週節奏：**Mon 攀岩 / Wed 攀岩 / Sat 攀岩**（中間隔休息，比原 Mon-Thu-Sat
  恢復更佳）
- Week 1 hangboard baseline 被跳過 → Week 2 週四（4/23）補做

---

## 核心原則（4 週貫穿）

1. **指力優先**（Lattice 共識，V5-V10 #1 limiter）
2. **每減 1kg ≈ +1.5% 有效指力**（Lattice *Stay Light or Get Strong?*）
3. **手指肌腱 48-72hr 才能重負荷**（Emil Abrahamsson、Hooper's Beta 共識）
4. **Rest more than you think**（Adam Ondra *The Nugget* Playlist #38）
5. **Progressive overload**：每週 +5% 負重或 +1 組，不 linear 就 plateau

## Week 2 — 建立 Baseline + PE 啟動 (2026-04-20 ~ 04-26)

### 新增 vs Week 1
- Monday 開始每週一次 **4×4 PE session**
- Thursday 補做 **hangboard baseline 測試**
- 每日加 **creatine 5g + Vitamin D3 2000 IU + Omega-3 2g**
- 每週 2 次（Thu + Sat 前）**collagen 15g + VitC 50mg** 於訓練前 30-45 min

### Monday 4/20 — Volume + 4×4 PE

**Volume 部分（60 min）**
- 熱身 + 暖身路線 15 min
- 攀 V3-V4 共 10-12 條（不需完攀每條，累計量）
- 休息 15-20 min

**4×4 PE 部分（25 min）**
- 選 4 條 V3-V4 難度（你的 flash-2 grade）
- 4 條連做 4 輪，**每輪之間只休攀爬時間 1:1**
- 輪與輪之間休 2-3 min
- 目標：**完成 16 次攀爬不掉落超過 2 次**

> 依據：Lattice *Training Climbing Power Endurance - 3 Mistakes* (Playlist #52)
> — 強調 PE 訓練必須 incomplete rest，不是 complete rest

### Thursday 4/23 — Hangboard Baseline + 輕量攀岩

**Hangboard max hang 測試（30 min）**
- 熱身：反覆在 20mm edge 上 bodyweight hang，6-10s × 3 組，逐漸適應
- Max hang protocol（Emil Abrahamsson）：
  - 20mm edge, half-crimp
  - 10 秒單次 max hang，休 **5 分鐘**
  - 逐漸 +2.5kg 至「能撐 8 秒但 10 秒掉」 → 該重量即為 baseline max
- 記錄最大負重（例：體重 + 10 kg = 82 kg total）

**之後輕量攀岩（30-45 min）**
- V3-V4 technique drill（silent feet、flag、drop knee 專注）
- 不攀高難度，手指已用

### Saturday 4/25 — Peak Day 1

**流程**：
1. 起床後：collagen 15g + VitC 50mg（泡水，不吃其他蛋白質干擾吸收）
2. 早餐：碳水 60-80g + 蛋白 20-30g，訓練前 **2 hr 停食**
3. 攀岩前 30-45 min：**咖啡 3-5mg/kg = 250-300mg**（約一大杯美式）
4. 熱身 20-30 min
5. 主體：V5 flash attempts → V6 project → 若狀況好嘗試 V7
6. 主攻項目：**`LAST CALL` cycle**（作為左-lock-off benchmark）
7. 攀岩後 30 min 內：30g protein + 60-80g carbs（如雞肉飯、蛋白奶昔 + 香蕉）

### 其他日
- Tue 4/21: 下肢（pistol squat, goblet squat, lunge）+ core + 拮抗 30-40 min
- Fri 4/24: Lock-off + compression（本週五 template）
- Wed 4/22 + Sun 4/26: 視 Mon/Thu 恢復狀況；**Wed 可以再攀**若感覺好，**Sun 必須休**

### Week 2 KPI 目標
- [ ] 體重 ≤ 71.5 kg（-0.5 kg）
- [ ] Hangboard baseline 數據建立（total weight at 8s fail）
- [ ] Mon 4×4 完成 16 次 ≥14 次
- [ ] Sat 至少完成 1 條 V6
- [ ] 平均睡眠 ≥ 7 hr

## Week 3 — 強度建設 (2026-04-27 ~ 05-03)

### 新增 vs Week 2
- Hangboard: **baseline + 5kg**（Thu）
- PE: **Mon 4×4 + Wed 補一次 linked circuit**（30 moves continuous）
- Sat 開始認真 project V7

### Thursday 4/30 — Hangboard +5kg

- 5 組 × 10 秒 at **baseline + 5kg**（約 85-90% max）
- 休息 **3 分鐘**（Emil protocol）
- 若第 5 組撐不足 8 秒 → 本週不加量
- 之後輕量攀 30 min

### Saturday 5/2 — Peak Day 2: V7 Project 首攻

- 若 `LAST CALL` 左 lock-off 已進步 → 全力衝
- 備選：`LAPUTA`（7A+/V7）用 compression 進步測試
- **最多 6-8 次 attempts**，之後肌腱疲勞風險過高

### Week 3 KPI 目標
- [ ] 體重 ≤ 71 kg（-1 kg 累計）
- [ ] Hangboard +5kg 完成 5×10s
- [ ] Sat 至少完成 2 條不同 V6
- [ ] V7 attempts: 至少其中一條到達 last move

## Week 4 — Peak Build (2026-05-04 ~ 05-10)

### 新增 vs Week 3
- Hangboard: **baseline + 10kg**（若手指乾淨；若刺痛 → 維持 +5kg）
- PE 加 linked circuits
- **Sat + Wed 都做 project V7**（雙峰日）

### Monday 5/4 — Volume + 4×4 + Linked Circuit

- 標準 4×4 ×1
- **+ 30 moves linked circuit on Moonboard / board**（中等難度，一鼓作氣）
- 總攀爬量比 Week 2-3 大

### Thursday 5/7 — Hangboard +10kg

- 同 protocol，重量再加
- **這是 4 週中最重的一次 hangboard session**

### Saturday 5/9 — Peak Day 3: V7 Send 嘗試

- 當週最認真 V7 attempt session
- 若體重 ≤ 70kg 且 hangboard +10kg 達標 → V7 send 機率最高

### Week 4 KPI 目標
- [ ] 體重 ≤ 70 kg（-2 kg 累計）
- [ ] Hangboard +10kg 完成 5×10s（或維持 +5kg 若肌腱訊號）
- [ ] V7 send **at least 1 條**

## Week 5 — Deload + Peak (2026-05-11 ~ 05-17)

### 整週邏輯
- Mon-Thu **減量 40%**（但不減強度）
- Fri 完全休
- **Sat-Sun 為真正的 peak 嘗試窗**

### Monday 5/11 — Light Volume
- 攀 V3-V5 共 6-8 條（量減半）
- 無 4×4 PE

### Tuesday 5/12 — 健身房 light
- 30 min，維持強度不加量

### Wednesday 5/13 — 輕度攀岩
- 嘗試 V5-V6 flash，感覺「指力回來了」
- 不上 V7，儲存體力

### Thursday 5/14 — Hangboard deload
- Baseline + 5kg（Week 3 水平），**3 組**（非 5 組）
- 短、精

### Friday 5/15 — 完全休息
- 按摩 / 瑜珈 / 散步
- 早睡 22:00

### Saturday 5/16 — **Peak Day: V7 Final Send**
- Collagen + VitC 訓練前 30 min
- 咖啡因 3-5mg/kg
- 完整 30 min 熱身
- `LAST CALL` 或 `LAPUTA` — 最多 6 attempts
- 若 Sat 沒 send → Sun 再一次機會

### Sunday 5/17 — Backup Peak / 慶功
- 若 Sat send 成功：輕鬆攀 + 拍影片記錄
- 若沒 send：再試 4-5 attempts（早上精神最好時）
- 計劃結束

### Week 5 KPI 目標
- [ ] 體重 ≤ 69 kg
- [ ] **V7 send 完成 ✓**

## 飲食執行表（每日）

### 攀岩日 / 訓練日

| 時段 | 內容 | 目的 |
|-----|------|-----|
| 起床 | 水 500ml + 若訓練在 early：collagen 15g + VitC 50mg | 肌腱、hydration |
| 早餐 | 碳水 60-80g（燕麥/飯/麵）+ 蛋白 25-30g（蛋/雞胸/希臘優格） | 訓練前燃料 |
| 訓練前 2hr | **停食**（避免攀岩時胃不適） | — |
| 訓練前 30-45min | 咖啡 3-5mg/kg（僅 Sat peak + 重要 hangboard 日） | 爆發力 + 疼痛閾值 |
| 訓練前 30-45min | collagen 15g + VitC 50mg（Thu + Sat） | 肌腱 collagen synthesis |
| 訓練中 | 水 + 少量 electrolyte（流汗多時） | 維持 performance |
| 訓練後 30min 內 | protein 30g + carbs 60g（奶昔 + 香蕉、或正餐） | 肌肉恢復窗 |
| 晚餐 | 蛋白 30-40g + 蔬菜 + 碳水 40-60g | 每日 125-145g 蛋白目標 |
| 睡前 1hr | casein / cottage cheese 20g（可選） | 夜間肌肉合成 |

### 休息日

- 碳水降到 2-3g/kg（= 150-220g），蛋白維持 125g+
- 總熱量 1700-1900 kcal（比訓練日 -200 kcal，創造 deficit）
- 仍吃 creatine + D3 + Omega-3
- 不吃 collagen + VitC（僅訓練前有意義）

### 每日 supplement

| 補充品 | 劑量 | 時機 | 為何 |
|-------|-----|-----|------|
| Creatine monohydrate | 5g | 任何時間 | 神經 + 爆發力 |
| Vitamin D3 | 2000-4000 IU | 早餐隨脂肪 | SG 室內工作者幾乎缺 |
| Omega-3 (EPA+DHA) | 2g | 早餐 | 抗炎、關節 |
| Magnesium glycinate | 300mg | 睡前 30min | 睡眠 + 神經放鬆 |
| 綜合維他命 | 1 | 早餐 | 保險 |
| **Collagen peptides** | **15g** | **訓練前 30-45min (Thu + Sat)** | **肌腱 collagen synthesis (Baar protocol)** |
| Vitamin C | 50mg | 同 collagen 時機 | collagen synthesis cofactor |

**絕對避免（5 週內）**：
- ❌ 酒精（抑制 mTOR、干擾肌腱修復）
- ❌ 訓練日 cold plunge / 冰敷訓練後 24hr 內（抑制 adaptive signaling）

## Deload 訊號（出現任一就減量）

- 指關節腫脹 / 僵硬 >24 hr
- 前臂酸痛 >48 hr 未退
- 睡眠品質突降 (<6 hr 連續 2 晚)
- 心跳晨起 >+10% baseline
- 任一手指刺痛

**處理**：立即休 2-3 天完全不碰手指；補眠 + 補水 + 蛋白質。

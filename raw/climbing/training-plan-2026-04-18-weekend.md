### 🧠 核心概念（C 與 D block 的運作邏輯）

> **一次讀懂「配額」「section work」「補量」是在講什麼。**

#### 兩個獨立 budget

今日你同時在消耗兩種資源，分開計算：

| Budget | 今日額度 | 由誰燒 | 恢復週期 |
|--------|--------|-------|---------|
| **手指 max-effort 配額** | **6 次 V7 attempts** | C block 的整條 attempts | 48–72 hr（肌腱修復）|
| **總訓練量 budget** | ~2.5 hr 全身能量 | A+B+C+D+E 全部 | 24 hr（肝醣/神經）|

→ C block 燒的是**手指配額**；D block 燒的是**總訓練量**（手指只剩 sub-max 負荷）。兩者互不替代。

#### 1️⃣「Attempt 配額」= 今日 V7 最高強度嘗試次數的硬上限

- **一次 attempt = 從 start hold 起攀到掉落為止**，不管掉在第 2 手還是第 8 手，都算 1 次
- **硬上限 6 次**（Week 1 tendon load 安全值；Week 4 可升至 8 次）
- **超過 → 手指屈肌腱（A2/A4 pulley）微損傷累積**，tendinopathy 風險指數上升
- **跨 project 共用配額**：LAST CALL 用 4 次 + LAPUTA 用 2 次 = 剛好 6，**不能換路線就歸零重算**
- 依據：Hooper's Beta 手指傷害預防系列；Lattice tendon load 48–72 hr 恢復共識

#### 2️⃣「Section work」不算配額

- **Section work = 只爬 2–3 手的局部 sequence**（例：從 move 4 hold 起，做 5→6 後跳下）
- 單次出力 5–10 秒（完整 attempt 是 40–90 秒）→ 手指負荷只有完整 attempt 的 1/5
- 作用是**學 beta / 練 sequence**，不是 max-effort output → 所以不計入 6 次配額
- **但不是無限**：section work 總時間 ≤ 20 min，且任何刺痛立刻停
- 依據：Lattice *How To Project Hard Boulders*；Adam Ondra project 策略 = full attempts + isolated sections 交替

#### 3️⃣ Active recovery 攀岩（V3-V4 慢攀）不算配額

- Sub-max 攀爬 → 手指循環用，非 max output
- 夾在高強度 attempts 之間可以 **降低前臂 pump，維持後續 attempt 品質**

#### 4️⃣「補量 V5-V6」= 用 sub-max 難度刷總攀爬量

- **為什麼 D block 需要**：V7 配額用完 → 手指不能再 max-effort，但身體還有總訓練量預算（腳步、body tension、讀線）
- **為什麼鞏固 V5-V6**：攀岩進步不只靠 max-effort，也靠 sub-max volume；本週目標穩定 V6 + touch V7 → V5-V6 要有足夠完攀量
- **為什麼需要指力 ≥ 50% 才做**：指力 < 50% 時 V5 動作品質崩 = 練壞動作 + 增加受傷；這時直接跳 E 收操更對
- **D 永遠是 bonus，不是 must-do**
- 依據：Lattice *How To Write Your Own Training Plan* — sub-max volume 是 V5→V7 transition 的第二大變數（次於指力）

#### 判斷流程（一張決策圖）

```
C block 跑完
   ↓
手指配額用完？
   ├─ 是 → 檢查總訓練量還剩多少
   │        ├─ 還有 ≥ 50%：做 D block（V5-V6 補量）
   │        └─ < 50% 或手指刺痛：跳 D，直接 E 收操
   └─ 否（因前臂 pump / 警訊提早停）→ 同上（總訓練量邏輯）
```

**一句話總結**：把最珍貴的資源（手指）放在最高價值的事（V7 project），剩的能量給鞏固性工作（V5-V6 量）。

##### C1. Project 主攻流程

| # | 動作 | 組 × 次 | 休息 | 備註 |
|---|-----|-------|------|-----|
| C1a | Attempt #1（整條從 start hold 起攀）| 1 次 | 5 min | 記錄：到第幾手 / 失敗原因 |
| C1b | Attempt #2（重試整條）| 1 次 | 5–6 min | 若 #1 進到 move 4+，#2 目標再前進；若 #1 就掉 → #2 改做 section |
| — | **Active recovery**：V4 flash 1 條 | 1 條 | 3 min | 手指循環，非配額 |
| C1c | Section work：分段 break down 難點 | 3–4 次 short burst | 2 min/次 | 只做卡住的 2-3 手 sequence，不整條；**不算主 attempts 配額** |
| — | **Active recovery**：V3 慢攀 1 條 | 1 條 | 3 min | 恢復 |
| C1d | Attempt #3（整條 link 嘗試）| 1 次 | 6–8 min | 最重要一次 — section work 後的 link |
| C1e | Attempt #4（視 #3 狀態）| 1 次 | 6–8 min | 若 #3 突破新高點 → 再衝；若停滯 → 考慮切備選 |

**切 project 判定（C1d 之後）**：
- C1d 有進到 move ≥ 6 且 link 感覺接近：繼續 C1e 同 project
- 連續 3 attempts 卡同一手，且二頭酸：切備選 project，剩 2 attempts 給備選
- 前臂 pump > 7 或手指警訊：**立刻進 D block**，不用完配額

| C2a | 備選 project attempt | 1–2 次 | 6 min | Fresh project，只看前段 sequence，不求完攀；**共用同一個 6 次配額**（見 §核心概念 1️⃣）|

> **依據**：Adam Ondra *Most Climbers Should Train LIKE THIS* — project attempts 需 full recovery (5+ min)；Lattice *How To Project Hard Boulders* — section work 不算 main attempts；Hooper's Beta — Week 1 tendon load 上限 6-8 attempts

**執行條件**：C 後指力主觀 ≥ 50%；若指已死 / 刺痛 / pump 不退 → 跳過直接 E（D 是 bonus，不是 must-do）。

## Self-reported failure modes

1. **續航力不足 (Power endurance deficit)**
   - Single moves OK, broken-down sequences OK
   - Continuous linking of moves → pump failure before top
   - Classic power-endurance (PE) limiter pattern

2. **L-sit / 鎖定 (Lock-off) 力道不夠 — 左側尤甚**
   - Cannot hold L-shape static position for left-right weight transfer moves
   - Left arm lock-off markedly weaker than right → asymmetry

3. **非垂直線上的抓握力不足 (Off-axis grip weakness)**
   - Holds fine when pulling straight down on a hold
   - Fails when force vector is off-axial (sloper/pinch directions, gaston, side-pull)
   - Some holds "totally cannot grip" on V7 problems

4. **二頭內拉力量嚴重不足 (Bicep-dominant inward pulling / compression)**
   - Compression + close-body pulls fail
   - Example: `LAST CALL` (Moonboard 40°, 7A+/V7, all left-hand holds) — final
     moves require powerful bicep-driven inward pull off the wall

# 2026-04-27 (週一) 疾病急性期 day 2 — 全休

> **🚨 疾病覆寫延續中**：原本今日 = Week 3 Day 1 Branch A（Volume + 4×4 PE 正式啟動）—— **全部取消**，因為 4/26 晨起確認 below-neck 症狀（咳嗽 + 鼻塞 + 疑似支氣管炎）。今日 = 病期 day 2，仍在 Eichner neck-check fail 區間，訓練 0 容忍度。
>
> **總時長**：0 min 訓練；100% 臥床 + 補水 + 補眠 + 症狀監測
> **類型**：疾病急性期 day 2 —— 手指 0 / 上肢 0 / 下肢 0 / 心肺 0
> **特化調整**：本日重點 = **判斷病程方向**（day 1→day 2 症狀變嚴重 / 持平 / 改善）—— 這個讀數決定 Tue 04-28 走 Tier A/B/C
> **本週實際節奏**：Sun 04-26 病期 day 1 ✔（全休）→ **Mon 04-27 病期 day 2（今日）**→ Tue 04-28 三階決策日
> **恢復脈絡**（雙線）：
> 1. 訓練面：T+2 from Sat 04-25 V3-V5 volume；訓練負債已清零，現在純粹等病好
> 2. 疾病面：day 2；典型支氣管炎 day 2-3 為症狀高峰，day 4-5 開始改善；若 day 2 已比 day 1 改善 → 病程偏輕
> **⚠ 核心紀律**：
> - 病期最常見錯誤 = 「我覺得好一點了，動一下試試」 → 直接打回 day 1 重來，且增加 myocarditis / 支氣管炎遷延風險
> - 「症狀改善」的 threshold 是 **24 hr 完全無症狀**，不是「今天比昨天好」
> - 即使你今天感覺好 80%，今日仍是全休 day 2

## A. 唯一允許的活動清單（同 04-26）

> 依據：Eichner neck-check rule（*Physician and Sportsmedicine*）；ACSM viral infection + exercise myocarditis 共識；Lattice *Should You Train When Sick?*（Tom Randall）—— below-neck → 完全停訓直至 24hr 無症狀。

## B. 飲食（病期 maintenance，不 deficit）

| 時段 | 內容 | 目的 |
|-----|------|-----|
| 起床 | 溫水 500ml | Hydration |
| 早餐 | 軟食碳水 60-80g + 蛋白 25-30g（粥 / 燕麥 / 蛋）| 能量 + 易消化 |
| 中餐 | 蛋白 30-40g + 蔬菜 + 碳水 50g | 主餐 |
| 晚餐 | 蛋白 30-40g + 蔬菜 + 碳水 40-50g | 不縮 |
| 全日 | 水 ≥ 3 L、總熱量回 maintenance（~2200-2400 kcal）| 免疫優先 |
| 補充 | Vitamin C 500mg-1g/日（病期短期高量 OK）+ Zinc 15mg/日 | 病期共識做法 |
| ❌ 禁 | 酒精、咖啡因 > 14:00、deficit | — |

> 依據：Cochrane review on Vitamin C for common cold（Hemilä & Chalker, 2013）—— 病程中高量 VitC 可縮短病程；Zinc lozenge meta-analysis（Singh & Das, 2013）同向。

## 刻意避開的內容（每項含原因）

- ❌ **任何訓練（攀岩 / 健身 / hangboard / core / 散步 > 5 min）** —— 病期運動會延長病程 + myocarditis 風險
- ❌ **「測試一下身體」** —— 即使 push-up 5 下、輕度伸展長 > 5 min 都算；day 2 是常見的「假改善」點，不要被騙
- ❌ **看訓練影片 / Moonboard / 5/16 milestone 規劃** —— 心理上也是負荷，本日 100% off
- ❌ **深呼吸 / 冥想式呼吸**—— 誘發咳嗽循環
- ❌ **泡澡 > 10 min / 蒸氣浴**—— 加重支氣管刺激
- ❌ **NSAIDs（ibuprofen / naproxen）**—— 用 acetaminophen 退燒；NSAIDs 病毒感染期證據混雜
- ❌ **熱量 deficit**—— 不變，病期暫停
- ❌ **熬夜（看劇 / 滑手機到 23:00 後）**—— 免疫修復賣血給娛樂

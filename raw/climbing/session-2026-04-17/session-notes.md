## Observed failure modes / rate-limiters

1. **Bicep / elbow-flexor chain is the cross-block bottleneck.** Typewriter
   (B4) near-limit on set 1 is consistent with B1–B3 having already drained
   the shared bicep/brachialis pool. This is the *same* muscle group flagged
   in the 2026-04-15 compression diagnosis — two independent sessions now
   point at it. Training implication: the asymmetric-pull programming is
   loading the right chain; dosage for this muscle group is now the
   constraint, not the exercise selection.

2. **Energy state directly gated tail-block output.** Fasted + coffee →
   post-B hunger → C executed at reduced intensity, D dropped. Not a
   strength-limiter question; a fueling question. Next gym day, carb intake
   90–120 min pre-session to preserve the C+D window.

3. **Rest discipline extended wall-clock time.** 2–3 min inter-set,
   5–6 min inter-exercise. Preserved force quality per set but pushed
   session to ~90 min — worth it for the B block (the point of today was
   quality on unilateral work), but the long transitions compounded the
   fueling gap.

## What this validates / invalidates (vs. plan)

- ✔ **Validates**: unilateral offset / 90°-pause / single-arm lock-off /
  typewriter as the right battery for left-side asymmetry. User felt
  left-side work was productive, not just painful.
- ✔ **Validates**: correct call to skip max hangboard today. T+2 finger
  recovery still intact going into tomorrow's Peak Day.
- ⚠ **Revises**: B-block is currently dose-heavy enough that C3/C4
  (pinch + curl) should either (a) be shortened or (b) swap a day so the
  shared bicep/brachialis chain isn't being hit 4th in the same session.
- ⚠ **Revises**: sessions following fasted day should pre-fuel carbs; do
  not schedule D (core) last in a fasted gym day.

## Prescription carried forward

1. **Keep** B-block structure (offset + 90° pause + single-arm lock + typewriter)
   for the remaining 4 weeks. It worked.
2. **Move** pinch-block work either earlier in the session (before B) or
   to Tuesday gym day, so the bicep chain isn't the 3rd–4th load of the
   same block.
3. **Fuel rule**: on Fri gym, carbs 90–120 min pre-session if fasted since
   lunch. Non-negotiable if D (core) is in the day's plan.
4. **Core (D) relocation**: consider moving Core to a separate morning slot
   or pairing it with Tue gym day rather than end-of-Fri.
5. **Sat Peak Day cue**: today's left lock-off work was real load — expect
   slight residual fatigue on LAST CALL benchmark tomorrow. Don't over-read
   if first attempts feel heavy.

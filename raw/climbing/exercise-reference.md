# 動作參照手冊

For tonight's plan ([training-plan-2026-04-17-fri.md](training-plan-2026-04-17-fri.md)),
以下是每個不熟悉動作的圖解 + 教學影片。大多來自 Eric Hörst (trainingforclimbing),
Lattice Training, Hooper's Beta 等公認權威。

---

## Block B — Lock-off 強化系列

### B1. Offset Pull-up（不對稱引體）

**中文別名**：毛巾輔助引體、Uneven-Grip Pull-up

**動作**：一手握桿，另一手握掛在桿上（向下約 30 cm）的毛巾/繩索，形成左右不等高。
主手承擔多數體重，輔助手只出少量力。

```
       ╔═══════╗
       ║  桿   ║  ← 主手握桿（弱側）
       ╚═══╤═══╝
           │
        毛巾 ↓
           │
       ╔═══╧═══╗
       ║ 握點 ║  ← 輔助手（強側，向下偏移 ~30 cm）
       ╚═══════╝
```

**教學**：
- [Develop One-Arm & Lock-Off Strength with Uneven-Grip Pull-ups — Eric Hörst](https://trainingforclimbing.com/develop-one-arm-lock-off-strength-with-uneven-grip-pull-ups/)
- [How to One-Arm Pullup (16 Levels) — Hooper's Beta](https://www.hoopersbeta.com/library/one-arm-pull-up-progressions)

**關鍵提示**：主手肘下壓、肩胛先啟動（不能只靠手臂拉）；毛巾手保持 30% 以下出力，感覺
弱側多承擔。若左側弱，左手握桿、右手握毛巾。

---

### B2. Pause Pull-up at 90° (90° 暫停引體)

**中文別名**：暫停鎖定引體

**動作**：標準引體向上過程中，在肘關節 90° 時暫停 2–3 秒（上升與下降各暫停一次），
再繼續到頂或下放。

```
 ↑ 頂點 (elbow 完全收)
 │
 ├─ 暫停 2–3s @ 90°  ◀── 鎖定訓練點
 │
 ↓ 全臂伸直 (hang)
```

**教學**：
- [Training: Perfect Pull-Ups for Climbing Strength — Climbing.com](https://www.climbing.com/skills/training-perfect-pullups-for-climbing-strength/)

---

### B3. One-Arm Lock-Off Hold (單臂鎖定靜態，毛巾輔助)

**中文別名**：單臂 L 型撐

**動作**：主臂彎 90° 或更深，身體懸掛在桿上；另一手抓毛巾或彈力帶協助分擔重量。
保持靜態 5–10 秒，主臂勿伸展。

```
     ╔═══════╗
     ║  桿   ║
     ╚═╤═══╤═╝
       │   │
     ▼ 90° └── 毛巾
       ✋主   ✋輔
       │
      身體懸吊
```

**教學**：
- [This Home Training Program Will Have You Locking Off Like a Monster — Climbing.com](https://www.climbing.com/skills/training-plan-improve-your-lock-off-strength-at-home/)
- [Lock-off training (evidence-based) — Eva López, PhD](http://en-eva-lopez.blogspot.com/2013/05/lock-off-training-for-climbers-v-goal.html)

**關鍵**：弱側 3 組 × 8–10 秒，強側 2 組。毛巾只提供「剛好能撐住」的輔助 — 不要讓
輔助手完全承重。每週左右側差距縮小 = 成功。

---

### B4. Typewriter Pull-up (打字機引體)

**動作**：拉到引體頂點後，身體左右水平平移（像打字機），一邊手臂伸直、另一邊彎曲，
再回中、平移另一邊。

```
  左平移          中點           右平移
   ↖              ↑              ↗
  ✋─── 桿 ───✋    ✋─── 桿 ───✋    ✋─── 桿 ───✋
  伸    彎        彎     彎         彎    伸
     ↓               ↓                 ↓
    身體左            身體中             身體右
```

**教學**：
- [Typewriter Pull-ups — Training Beta](https://www.trainingbeta.com/media/pull-ups-typewriters/)
- [Learn The Typewriter Pull UP — Gornation](https://www.gornation.com/blogs/calisthenics-exercises/typewriter-pull-up)

**提示**：慢、可控；每側停 1 秒。6 次/組 = 左右各 3 次。

---

### (Bonus) Frenchies — 可選替代 B2

**動作**：單組內三次引體，分別在不同角度暫停：
1. 拉到頂 → 頂部停 5s → 放下
2. 拉到頂 → 下放到 90° → 停 5s → 下放
3. 拉到頂 → 下放到 120° (接近伸直前) → 停 5s → 下放

**教學**：
- [4 Pull-up Variations to Amp-Up Your Workouts (Frenchies 詳細說明) — Eric Hörst](https://trainingforclimbing.com/4-pull-up-variations-to-amp-up-your-workouts/)

**為何推薦**：一次覆蓋三個關鍵鎖定角度，是攀岩中公認最佳 lock-off 複合動作。

## Block C — Compression 系列

### C1. Narrow Supinated Chin-up (窄距反握引體)

**動作**：掌心朝自己（反握），雙手握距比肩窄（約 15–20 cm），做引體。
強調二頭肌、肱肌（brachialis）與 pecs。

**教學**：
- [4 Pull-up Variations to Amp-Up Your Workouts — Eric Hörst](https://trainingforclimbing.com/4-pull-up-variations-to-amp-up-your-workouts/)（同上篇，章節有 chin-up）

**提示**：下放 3 秒慢離心，避免反彈；可感受到二頭強烈收縮。

---

### C2. Dumbbell Pullover (啞鈴仰臥上拉)

**動作**：仰臥長凳，雙手握一支啞鈴上舉於胸前 → 慢慢下放過頭（啞鈴呈弧形到額頭上方）→
緩慢拉回起點。重點不是重，是感受 lat + pec + serratus 整合收縮。

```
起點 → 啞鈴正上方    中間 → 啞鈴過頭    終點 → 拉回胸前
  🟰                   ⤵                 🟰
  手臂直立            手臂弧形下放       再收回
```

**教學**：
- [How To Do A Dumbbell Pullover — Athlean-X](https://learn.athleanx.com/articles/back-for-men/how-to-do-dumbbell-pullovers)
- [Dumbbell Pullover Guide — MasterClass](https://www.masterclass.com/articles/dumbbell-pullover)

**提示**：手肘保持微彎固定；下背不離凳；核心收緊。攀岩者重點在 lat–serratus–pec 串聯，
這正是 compression（二頭內拉）的主動肌群。

---

### C3. Pinch Block (捏力磚)

**動作**：兩手同時捏住一塊矩形木塊/鋁塊（Lattice Quad Block 是標準規格），
垂直提起固定重量 15–20 秒。訓練純捏力。

**教學**：
- [Get Stronger With Pinch Block Training — Gripped Magazine](https://gripped.com/indoor-climbing/get-stronger-with-pinch-block-training/)
- [Effective Pinch-Grip Repeater Training — Eric Hörst](https://trainingforclimbing.com/pinch-grip-repeater-training-for-climbers/)
- [Lattice Quad Block 官方](https://latticetraining.com/product/quad-block/) — 標準訓練工具
- Playlist #16: *Incredible Sloper Strength with the Heavy Roller* (Lattice, 4:16)

**提示**：今日 3 × 15s，重量選「左右都能完成」。不求新高紀錄，求 compression 肌群啟動。
若健身房沒 pinch block，用兩塊啞鈴片（平面朝外）夾起來也能替代。

---

### C4. Supinated DB Curl (反握啞鈴彎舉 — 慢離心)

**動作**：標準啞鈴彎舉，掌心朝上（反握），下放時 3 秒慢離心。

**教學**：
- 標準動作，任何健身教學站皆可 — 重點是**離心速度**與**肌腱健康**，不是重量。

**為何這樣做**：二頭肌腱保養 + 補強，預防 compression 重量訓練誘發的肌腱炎（per
Hooper's Beta 肌腱健康指南）。

---

## Block D — Core + 軀幹張力

### D1. Front Lever Tuck Hold (前水平抱膝撐)

**動作**：懸吊桿上 → 把膝蓋收到胸前 → 身體向後旋轉讓背部平行地面 → 維持 15 秒。

```
  💪 桿 💪
   │   │
   │   │  ← 手臂直
   │   │
   └───┘
     ━━━━━━━  ← 背部平行地面
       │
      🦴 抱膝
```

**教學**：
- [How to Do a Front Lever — Climbing.com](https://www.climbing.com/skills/how-to-do-a-front-lever/)
- [Learning How to Front Lever for Climbing — Gripped Magazine](https://gripped.com/indoor-climbing/learning-how-to-front-lever-for-climbing/)
- [Front Lever Progression Guide in 8 Steps — A Shot of Adrenaline](https://ashotofadrenaline.net/front-lever-progression/)

**進階**：若 tuck 可穩定 20s，下週可進 advanced tuck（膝略伸）。

---

### D2. Hanging Leg Raise (懸掛舉腿)

**動作**：懸吊桿上，雙腿伸直，用核心（非髖屈肌）把腿舉到水平。

**提示**：腳舉起時骨盆要後傾（posterior pelvic tilt），不是單純抬腿。10 次/組，
慢下放（3 秒離心）。

---

### D3. Dead Bug (死蟲式 — 加輕啞鈴)

**動作**：仰臥，手腳各呈 90° 朝天花板 → 對角伸展（左手 + 右腿、右手 + 左腿交替），
下背全程貼地面。

```
起點：             動作：
      ✋ ✋           ✋—→         (左手伸)
      🦵 🦵              🦵←       (右腿伸)
         下背貼地       下背 **不能拱**
```

**教學**：
- [Dead Bug Exercise Tutorial — NASM](https://www.nasm.org/resource-center/exercise-library/dead-bug)
- [How to Master the Dead Bug Exercise — Men's Journal](https://www.mensjournal.com/health-fitness/how-to-do-dead-bugs)

**進階**：手持 2–5 kg 啞鈴或藥球增加抗伸展挑戰（anti-extension）。

---

### D4. Side Plank + Hip Raise

**動作**：側棒式起點（肘撐地），hip 下沉碰地 → 上抬至頂 → 連續上下 15 次 / 側。
訓練腹斜肌（obliques）與腰方肌，攀岩 tension 必備。

---

## Block E — 拮抗 + 收操

### E1. Scapular Pull-up (肩胛引體)

**動作**：懸吊直臂 → 不彎肘、只用肩胛下壓往後夾，身體上升 ~5 cm → 回起點。

```
起點：手臂直、肩胛放鬆      動作：肩胛下壓 + 後夾
    ┊                        ↓
    ┊←━肩胛上舉              ┊ ←━ 肩胛下壓
    ┊                         ┊
    🧍                        🧍 上升 5 cm
```

**教學**：
- [The Best Climbing Exercise You're Not Doing: The Scapular Pull-up — Eric Hörst](https://trainingforclimbing.com/the-best-exercise-youre-not-doing-the-scapular-pull-up/)
- [Scapular Pull-up Video — Eric Hörst](https://trainingforclimbing.com/video-the-best-climbing-exercise-youre-not-doing/)
- [Are You Missing Out On This Key Shoulder Exercise? — Gripped Magazine](https://gripped.com/indoor-climbing/are-you-missing-out-on-this-key-shoulder-exercise/)

**為何重要**：攀岩所有拉動作的啟動肌群，保護肩膀，預防旋轉肌腱炎。

---

### E2. Band Pull-apart (彈力帶水平開胸)

**動作**：雙手握彈力帶於胸前，直臂向兩側拉開至胸碰帶，肩胛後夾 → 慢回。

**教學**：
- [How to Do Band Pull Aparts — Your House Fitness](https://www.yourhousefitness.com/blog/exercise-tutorial-band-pull-aparts)
- [Band Pull Apart Video Guide — Muscle & Strength](https://www.muscleandstrength.com/exercises/band-pull-apart)

**提示**：後三角 + 中斜方啟動；慢 1–2 秒伸、停 1–2 秒。

---

### E3. Diamond Push-up (鑽石俯臥撑)

**動作**：雙手拇指食指貼在一起成鑽石形，正常俯臥撐動作。三頭 + 胸內側 + 前鋸肌啟動。

---

### E4. Rotator Cuff 外旋（熱身 A 也有）

**動作**：肘貼身體、前臂 90° 向外開（對彈力帶阻力）。2 × 15/side。

**教學**：
- Playlist #25: *Basic Wrist Stability Training for Climbers* (Hooper's Beta, 7:16)
- 同系列：*Fix Your Wrist Pain! Follow-Along Routine* (Tone and Tighten)

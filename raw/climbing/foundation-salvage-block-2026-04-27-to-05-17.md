## 核心原則（21 天貫穿）

1. **不冒進 return-to-train**：病後第一週嚴格 ≤ 50% volume（ACSM 共識）；第二週 70%；第三週才 90%
2. **基礎優先，attempt 機會主義**：V5 flash 多風格庫存堆量是核心；V6 attempts 是副產品，**不 project**
3. **Bicep chain 重新驗證**：04-21 acute event 後的 clean window 數據被病期凍結 → Salvage 期內必須重建 ≥ 3 次無訊號讀數才能碰 V6 compression 動作
4. **5/16 milestone ≠ 驗收**：能拼就拼，不能拼就承認，不傷不疲為硬底線（5/18 入職）
5. **不為 milestone 加碼**：原本 W5 是 deload + peak window；Salvage 版仍保留 deload，因為 5/18 後沒緩衝

**Phase 0 結束條件**：症狀完全消退 ≥ 24hr **AND** 已完成「下肢 / core 50% session」+「散步 30 min × 2」+「輕度技術攀 V3-V4 一場」三項各一次。

> 依據：ACSM return-to-train 50% volume 第一週規則；Lattice *Should You Train When Sick?*（Tom Randall）—— 三階段 ramp（cardio low → sport-specific easy → sport-specific moderate）。

### Phase 1 — Foundation Rebuild（Phase 0 結束 → 5/9 Sat，~7-10 天）

**目標**：V5 flash 多風格庫存重建到 ≥ 5 條（≥ 3 風格）；hangboard 接續 Thu 04-23 baseline 漸進到 +5kg；bicep chain clean window 重建 ≥ 3 次無訊號讀數。

#### 週節奏（不固定週幾，依病後 ramp 起點滾動）

| 角色 | 內容 | 量 |
|------|------|-----|
| 攀岩日 1（如 Mon）| Volume V3-V4 多風格 + V5 flash attempt（≤ 2 go per 條）| 60 min |
| 健身日（如 Tue）| 下肢 50-70% + core + **輕度上肢**（首次重新引入：lat pulldown 50% / row 50%；**禁 curl / 拮抗 deep work**）| 60 min |
| 攀岩日 2（如 Thu 或 Wed）| V3-V5 flash 多風格 + **bicep chain 觸診**（pre + post）| 70 min |
| Hangboard（如 Thu 或 Fri）| Bodyweight 10s × 5（baseline 接續），**先不加負重** | 30 min |
| 全休（如 Wed + Sat 或 Sun）| 散步 / 伸展 | — |

#### Bicep chain 重新驗證 protocol

- 每攀岩日 pre + post 觸診（二頭 + 肘內側，0-10）
- 連續 **3 次攀岩日 0 訊號** → chain clean window 重建 ✔ → Phase 2 可碰 V6 compression
- 任一次 ≥ 2 訊號 → 重置計數，回到 V3-V5 only 直到下次 clean

> 依據：[2026-04-22 debrief](../../wiki/derived/2026-04-22-bouldering-session-debrief.md) §6 Cross-session clean window 操作定義；user's synthesis post-04-26 illness 將 "2 sessions clean" 升級為 "3 sessions clean"，因為病期肌腱 deconditioning 加上 chain 病前已有 3 次獨立訊號，需要更保守 buffer。

#### Phase 1 結束條件

- ✔ V5 flash 庫存 ≥ 5 條（≥ 3 風格）累計
- ✔ Hangboard 至少 1 次 +5kg × 5×10s 完成（不需 +10kg）
- ✔ Chain clean window ≥ 3 sessions（含 bodyweight hangboard）
- ✔ 任一日訓練後晚間倦怠 ≤ 5/10 + 隔晨心率 ≤ baseline +5 bpm

任一條件未達 → Phase 1 延長至 5/12（5/16 前 4 天為硬底線，必須留時間 deload）。

### Phase 2 — Bridge to Milestone（5/10 → 5/16 Sat，7 天）

**目標**：1-2 場 V6 2-go attempt 體驗（庫存 V6 嘗試經驗）+ 5/16 一日 milestone + deload 收尾。

#### 週節奏

| 日 | 內容 |
|----|------|
| Mon 5/11 | 攀岩 light：V3-V4 flash 5 條 + V5 flash 2-3 條（**不碰 V6**）—— 維持手感 |
| Tue 5/12 | 健身 50%（下肢 + core + 輕度拉力 50%；禁 hangboard）|
| Wed 5/13 | 攀岩：V4-V5 flash 多風格 + **V6 2-go attempt × 2-3 條**（風格不同；每條 ≤ 3 go）—— V6 first exposure 在 Salvage cycle |
| Thu 5/14 | Hangboard deload：baseline + 0~+2.5kg × 3 組（非 5）+ 輕度技術攀 V3-V4 30 min |
| Fri 5/15 | **完全休息** —— 早睡 22:00；不看 5/16 beta；不外食酒精；早餐 carb load 開始 |
| **Sat 5/16** | **Milestone Day**（見下方）|
| Sun 5/17 | 補測 day（如 Sat 達 V6×1，可早上補一條不同風格 V6）+ post-cycle debrief 起草 |

> 依據：Lattice *How to Write Your Own Training Plan* —— peak day 前 2 天 deload；Adam Ondra *The Nugget* — 1 週 3 場攀岩 + 1 場 hangboard 為 working volume 上限。

## 5/16 Milestone Day 規格

### KPI（修正版）

- [ ] **V6 完攀 ≥ 2 條（不同風格，flash 或 2-go）** —— 修正自原 V6×3
- [ ] **V5 flash ≥ 3 條（不同風格）** —— 鞏固性指標
- [ ] **V7 attempt ≥ 2 次（機會主義，符合強項即試）** —— attempt-ready 指標
- [ ] **Stretch goal — V6 × 3 不同風格**（達成則寫入 debrief 為超預期；未達**不算 fail**）
- [ ] **0 受傷 / 0 chain 急性訊號** —— **硬底線**，5/18 入職前不能留傷

### 流程（與原 5 週計劃 W5 Sat 相同骨幹，但 V7 配額降為 2 次、V6 配額 4 go × 2 條）

1. 起床：水 500ml + collagen 15g + VitC 50mg
2. 早餐：碳水 60-80g + 蛋白 20-30g，訓練前 2hr 停食
3. 訓練前 30-45 min：咖啡 3-5 mg/kg
4. 熱身 25-30 min（V3 → V4 → V5 階梯）
5. 主體：
   - V4-V5 flash 熱手 4-5 條
   - **V6 完攀 2 條（目標）**：每條 attempt 上限 4；總 V6 attempts ≤ 8
   - **V7 機會主義 attempt ≤ 2 次**（單條，符合強項）
6. 收操 + 30g protein + 60-80g carbs

### 4 個 stop-loss 訊號（同原 peak day）

- 任一手指刺痛 / 腫脹
- 肱二頭 / 肘內側 sharp tug
- 連續 3 次 V6 嘗試明顯退步
- 前臂 pump > 7/10 持續 > 10 min

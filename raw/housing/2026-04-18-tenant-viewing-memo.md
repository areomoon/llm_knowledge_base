## ⚠️ C. 必須明講的規則（按優先順序）

### 🔴 1. **不報戶籍 / 不設戶口** — **最重要**

- **時機**：不要一進門就講。等他表現有興趣、接近談租金條件時再提
- **說法範本**：
  > 「這邊有個小 house rule 先讓你知道 — 我們這個租金是**不遷戶籍**的版本。
  > 如果你們有需要報戶的話我們再另外談，但這個租金條件是維持在**不設戶**的前提。」
- **預期反應**：
  - 多數租客：「OK 沒問題」→ 通過
  - 少數會問：「為什麼？」→ 不用解釋稅務，只說「這是我們的原則，跟租金條件綁在一起」
  - **🚩 紅旗**：堅持要設戶 + 不願加價 → 謝謝再聯絡

### 🟡 2. 租期

- 目標：**簽 1 年，內部偏好 2 年**（長穩）
- 問他：「你們預計這個地方想住多久？」

### 🟡 3. 押金

- 業界慣例：**2 個月**（不是 1 個月）
- 砍到 1.5 個月可協商，1 個月是紅旗（現金流緊）

### 🟡 4. 其他細項

- [ ] 水電瓦斯自付（確認）
- [ ] 網路 / 第四台是否包含？（你家現況）
- [ ] 寵物：允不允？
- [ ] 抽菸：室內禁菸？陽台可？
- [ ] 訪客過夜？（有沒有要限制）
- [ ] 能否分租 / 加室友？（一男一女 → 避免之後偷加室友）

## 🚩 E. 紅旗 — 看到這些直接不租

- ❌ 職業說不清楚 / 問細節就迴避
- ❌ 不願提供名片 or 在職證明
- ❌ 堅持要遷戶籍但不願加價
- ❌ 押金要低於 2 個月
- ❌ 前一個房子住不到 6 個月
- ❌ 一直抱怨前房東
- ❌ 要求養 2 隻以上寵物
- ❌ 當場就想簽 + 押金不到位（急太奇怪）
- ❌ 問很多「你們什麼時候回台灣 / 會不會突然回來」（可能想搞事）

## 📋 F. 明天時間序建議

| 時段 | 動作 |
|-----|------|
| 看房前 1 hr | 再看一次租客 LINE 訊息 / 背景 |
| 客人到 | 簡短寒暄 → **請他們自己先走一圈**（別急著推銷） |
| 他們看 5-10 min | 跟在後面，**問他偏好**、他看到什麼有反應就著墨 |
| 邊走邊介紹 | 賣點（全聯 / 採光 / 開火 / 住戶單純） |
| 坐下來 | 聊職業 + 之前看過什麼 + 入住時間 |
| **接近收尾** | 講規則（**戶籍、押金、租期**） |
| 臨走前 | 「有想法的話 LINE 我們，明後天給個回覆」— 不逼成交 |

## 💰 G. 若雙方有意 — 後續三步

1. **當晚** LINE 謝謝來看房 + 問「有任何問題歡迎再問」
2. **48 hr 內**確認是否要租（給 deadline 避免他還在比價）
3. **決定租就約簽約** —
   - 簽約時收**押金 2 個月 + 首月租 = 3 個月總額**
   - 帶身份證影本 + 在職證明
   - 雙方填好物件點交清單（家具家電 item by item 簽名）

## 🗣 H. 關鍵對話範本（背一下）

**戶籍規則**（最怕講不好）
> 「這個價格是**不遷戶籍**的版本，這是我們的原則。如果你們一定要報戶，那我們要另外談一個不同的條件。」

**問職業**
> 「業務經理哪方面的？我自己也待過業務，好奇問一下。」

**問前一家為什麼搬**
> 「原來住哪邊？怎麼會想搬過來？」

**收尾（不逼成交）**
> 「我們再各自想一下，LINE 保持聯絡。我們這邊會有幾組看，但你們要是決定了就儘早告訴我們，我們給你優先。」← 製造緩衝 + 微鬆競爭感

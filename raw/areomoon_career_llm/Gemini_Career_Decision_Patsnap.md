# Gemini 諮詢：Patsnap 職涯決策全紀錄

> 來源：Gemini 對話 (2026-04-08)
> 諮詢主題：Coupang 台北 vs Patsnap 新加坡 vs 大廠搜推 Offer 的職涯抉擇

---

## 背景

- 目前在台北 Coupang，主要做 maintain pipeline + 修 bug，薪資約 130k SGD
- 收到 Patsnap 新加坡 Offer：Algorithm Expert，年薪 202k SGD（月薪 13.5k + 1 個月 AWS + 2 個月 Target Bonus）
- 同時考慮過 Binance、TikTok 等搜推方向的機會
- 個人狀況：32 歲，持有新加坡 PR，計劃近期登記結婚，無小孩，熱愛攀岩

---

## 一、Patsnap 職位分析

### 與 Hiring Manager（Huwei 哥，前同事）的溝通要點

1. **崗位職責校準**：
   - 短期內不負責 Agent 開發（主管認為 Agent 偏工程，對資深算法是浪費）
   - 核心工作：搜索算法（召回與排序）、Embedding、RAG、SFT
   - 團隊偏好全棧模式，成員兼顧設計/前端/後端/測試

2. **主管對 Agent 的冷思考**：
   - Agent 層在垂直行業會變得「薄」，核心在 Context Engineering 與後端穩定性
   - 資深算法做 Agent 是大材小用，應由 SDE 負責
   - 建議發揮搜索與推薦 7 年經驗，結合 SFT/RL

3. **團隊文化**：
   - 精簡團隊，「在精不在多」
   - 高度依賴 AI 工具（Cursor, Claude, Figma AI）
   - 產品上線前有高強度「趕工期」，核心成員常主動討論到深夜
   - 最終目標：建立「材料界的 Alpha」，往 RL 與模型訓練發展

### HR 通話確認事項

- **薪資結構**：13.5k SGD 為 Gross Salary（稅前），CPF 另計
- **醫療保險**：試用期即生效（含 GP、Specialist、牙科、住院手術）
- **試用期**：6 個月，期間福利與正式員工一致（除保險需再確認）
- **年假**：14 天，可結轉至次年 6 月 30 日
- **調薪**：一年兩次（2 月普調 + 8 月晉升/特殊調薪）
- **辦公模式**：RTO 週一至週五，無 WFH，地點 75 High Street
- **無 Sign-on Bonus、無股權**，全現金 Package

## 二、決策分析框架

### 三條路徑對比

| 維度 | 留在 Coupang（台北） | 接 Patsnap（新加坡） | 拼大廠搜推 Offer |
|------|---------------------|---------------------|-----------------|
| 薪資 | ~130k SGD，穩定 | 202k SGD，偏低但高於現職 | 300-350k SGD（E5 級） |
| 技術成長 | Pipeline 維護，停滯 | RAG/SFT/RL 實戰，高成長 | 成熟搜推，邊際遞減 |
| 履歷價值 | 大廠光環但 Low Impact | AI 核心落地經驗 | 大廠 E5 + Scale 經驗 |

### 同儕對比焦慮

- Grab 前同事分別在字節（電商推薦）、Apple（短影音推薦轉入）、Meta（MLOps）
- 均升至 E5 級別，年薪 300-350k SGD
- 差距來源：他們在「平台溢價」成熟賽道變現，自己在做技術轉型投資

## 三、決策結論與行動

### 最終決定：口頭接受 Patsnap Offer

### 決策理由
1. 32 歲是技術轉型最後窗口，在小孩出生前完成高強度衝刺
2. Pipeline 維護經驗不足以敲開一線 AI 公司的門
3. 垂直領域 RAG/SFT 實戰經驗是「帶薪讀博」，技術複利高
4. PR 身份提供隨時跳槽的自由度，不被簽證綁定

### 識別的風險
1. **薪資錨點風險（最擔心）**：202k 的 Base 可能影響未來談薪
   - 對策：用 Expert 職級對標 Staff Level；累積可量化戰績；利用 PR 創造競價環境
2. **技術孤島風險（次擔心）**：材料科學太 Niche，經驗難遷移
   - 對策：使用主流工具（LangGraph, RAGAS, vLLM）；聚焦通用方法論而非領域特例
3. **戰略誤判風險**：公司可能縮減 AI 投入
   - 對策：前三個月觀察 GPU 預算與產品規劃
4. **RTO 損耗風險**：全進辦公室 + 高強度 vs 新婚生活平衡
   - 對策：利用辦公室靠近岩館的地理優勢

### 短中長期規劃

**短期（入職前 1 個月）**：
- 身心重開機，攀岩、陪伴侶、辦登記
- 低成本預熱：LangGraph、LlamaIndex Advanced Retrieval、LLM for Scientific Literature 論文

**中期（入職後 6-18 個月）**：
- 攻克「多模態/跨段落證據鏈整合」，建立 High-Precision RAG 技術
- 洗掉 SDE/Pipeline 標籤，LinkedIn 關鍵字轉向 LLM/RAG/Agent/SFT
- 利用 PR 社交優勢，參加新加坡 AI 社群活動

**長期（18-36 個月）**：
- 帶著 Expert 頭銜 + AI 實戰作品集回歸大廠
- 目標對標 E6/Staff Level，薪資 350-450k SGD
- 瞄準 Meta GenAI、Binance 算法核心組、或 OpenAI/Anthropic 的落地團隊

## 四、關鍵洞察

1. **搜推是底座，RAG/Agent 是入場券**：傳統搜推正被 LLM 全線重構，未來面試不只問召回排序，更問如何用 LLM 增強
2. **垂直領域 > 通用領域**：通用大模型紅利期已過（2021-2022），現在是垂直行業應用爆發期
3. **Agent 的本質是工程**：對資深算法來說技術較淺，RAG/SFT/RL 才是算法深度的體現
4. **Niche 可以降維打擊**：解決材料科學文獻的極端難題後，回頭看金融/法律場景會更容易
5. **薪資差距靠稀缺性追回**：不是靠努力，而是靠「垂直領域 AI 落地專家」的不可替代性

---

## 五、面對主管的回答策略

當被問「為何捨棄大廠搜推 Offer」時：
- 核心邏輯：「搜推是優化過去，GenAI 是創造未來」
- 強調範式轉移：推薦系統邊際效益遞減，下一個爆發點在 Reasoning 與知識整合
- 強調數據密度：材料科學的髒數據難度遠高於通用 Chatbot
- 展現 All-in 決心：「技術成長潛力 > 短期 Package 差距」
